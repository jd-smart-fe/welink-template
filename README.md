# welink-template

> 微联项目模板库

## Install Template

推荐使用 [welink-cli](https://github.com/jd-smart-fe/welink-cli) 工具安装模板

```bash
$ npm install -g welink-cli
$ welink init  <project>
$ cd <project>
$ npm install
$ npm run dev
```
注意:  在执行 ``welink init <project> `` 的时候会提供自动选择模板
